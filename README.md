# React Store SPA

A simple React Single Page Application (SPA) built using React, Vite, and React Router.

## Features

- Home Page
- Shop Page
- Product Details
- React Router
- Dynamic Routing
- Context API
- Add to Cart
- Cart Count
- Checkout Page
- Local Storage Persistence
- Guest Login
- Protected Checkout Route

## Technologies

- React
- Vite
- React Router DOM
- Context API
- JavaScript
- CSS

## API

https://dummyjson.com/products

## Installation

```bash
npm install
npm run dev
```
